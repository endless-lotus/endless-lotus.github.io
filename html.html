HTML Knowledge

Semantic

To setup a webpage with how your mind maps it with whatever tools(semantic elements) fit best. This makes it easier for screen readers to process information and style with css with adequete groupings.

Semantic Elements Uses/Defintion
nav Defines navigation links.
header Specifies header of document or section.
main Specifies the main content of the document.
footer Defines the footer for the docuemnt or section.
section Defines section in a document.
article Defines independent, self-contained content.
details Defines additional details that the user can view or hide.
summary Defines a visible heading for a details element that can be clicked on to view/hide the details.
figure Specifies that an illustration, diagram, photo, code, etc. will be shown.
figcaption Caption for the image within the figure.
mark Defines highlighted text.
time Defines a date/time.
aside Defines content aside from the page content.

Details & Summary

        
        <details>
          <summary>Details</summary>
            There's a bunch of details here...
        </details>

The above code will output this to the page:

Details There's a bunch of details here...

Non-Semantic

div and span both have their wide plethora of uses. However, when setting up a page to read nicely a repition of div's and span's makes reading the code and changing later pesky.

Elements Uses/Defintion
div Does not represent anything as a "pure container" it has no effect on the effect on the container or layout until it is styled by CSS.
span An inline container used to style text with CSS or manipulate with JavaScript. Same as div element but inline instead.

Diplaying Code

To dispaly code without HTML automatically running it you have to jump through some hoops. First you must encase the area with the pre tag and wrap again in a second layer with the code tag. Then you must encode your code so it will show properly to the screen. You can do this easily with an encoder!

Tags Uses/Defintion
pre Defines preformatted text, keeps all the whitespace the same in the allocated area between the tags!
code Defines a piece of computer code and will display text in monospace font. 
        
        <pre>
          <code>
            Put Your Desired Code Here Remember to use the encoder!
          </code>
        </pre>
