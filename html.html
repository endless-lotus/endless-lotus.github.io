HTML Knowledge

Semantic

To setup a webpage with how your mind maps it with whatever tools fit best.

  • nav

    • Defines navigation links.

  • header

    • Specifies header of document or section.

  • main

    • Specifies the main content of the document.

  • footer

    • Defines the footer for the docuemnt or section.

  • section

    • Defines section in a document.

  • article

    • Defines independent, self-contained content.

  • details

    • Defines additional details that the user can view or hide.

  • summary

    • Defines a visible heading for a details element that can be clicked on to view/hide the details.

  • figure

    • Specifies that an illustration, diagram, photo, code, etc. will be shown.

  • figcaption

    • Caption for the image within the figure.

  • mark

    • Defines highlighted text.

  • time

    • Defines a date/time.

  • aside

    • Defines content aside from the page content.
